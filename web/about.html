About Fantasiagenesis

Fantasiagenesis is an experiment in turning thoughts into structured, navigable “dirt boxes” of data, prompts, and analyses. The project stitches together backend tools for creating boxes and running rich analyses, plus simple web views for exploring them.

Highlights:

  • Create and manage boxes of dirt, then run many analysis prompts over each box.
  • Queue-powered backend for LLM-derived analyses.
  • Simple web front-ends: box list, box detail, queue monitor, and read-only views.

Source code: github.com/b0risfosso/crayon
