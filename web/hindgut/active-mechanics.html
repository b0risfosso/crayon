Pre-Scripted Scenarios

  • Open-loop pulses: drift, no ratchet
  • Closed-loop curvature feedback: stabilization + ratchet, kernel biphasic
  • Wrong-sign / excessive delay: destabilization & oscillations
  • Strain sensor: stabilization with different kernels, predictive scores

Readouts

  • Global shape error over time
  • Variance reduction metric
  • Ratchet index
  • Cross-correlation tension vs light
  • Kernel predictive R²

Load-Bearing Test

Run 90s demo: open-loop drift → closed-loop stabilization → kernel inference → wrong-sign failure. Validation if experts recognize pulse-to-ratchet, tune gains/delays intuitively, accept kernel predictive power, and propose concrete perturbations (e.g., ROCK inhibition).
