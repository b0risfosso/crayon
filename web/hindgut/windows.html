Windows of Irreversibility — Symmetry‑Breaking Rotation / Seed

Problem → Unknown if transient heating/cooling leaves lasting chirality bias • Objective → Quantify depth & duration of chirality memory after brief thermal biases • Link → Double‑pulse protocols + path dependence fits to non‑Markovian memory kernels

Core intent: A brief thermal bias may leave a dose‑ and time‑limited memory of chirality such that outcomes depend on history, with a finite window beyond which a counter‑pulse cannot reverse the twist.

Panel B — Outcome Cards (Archetypes)

  • Reversible: counter‑pulse always flips outcome; no memory.
  • Soft Memory: flip probability decays with recovery; partial hysteresis.
  • Locking: sharp boundary beyond which flips fail; irreversible after window.
[Three columns: rotation vs time traces + final sign + confidence stamp]

Panel E — State Machine Sketch

States: Uncommitted → Soft‑Biased → Locked. Arrows: bias pulse drives Uncommitted→Soft‑Biased; decay loop; crossing bias×time threshold during recovery causes Soft‑Biased→Locked. Counter‑pulse resets before lock; fails after lock. Clock icon shows closing window.

[Three‑state diagram + arrows + clock]

Panel F — What‑If Narrative Sliders

  • Increase recovery → reversible: always flips; soft: flip chance ↓; locking: crosses cliff.
  • Weaken counter‑pulse → reversible: still flips; soft: depends on ΔT×t; locking: no effect post‑lock.
  • Move earlier in development → expands window (soft); no effect if instant locking.
