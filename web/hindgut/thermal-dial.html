Problem → Raw recoil speeds vary with temperature, obscuring tension • Objective → Normalize recoil by viscosity • Link → Ablation recoil × viscosity proxy = tension metric

Core intent: test whether multiplying recoil speed by a local viscosity estimate collapses temperature differences into a stable, comparable pre-stress proxy.
