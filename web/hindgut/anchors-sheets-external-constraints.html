Anchors, Sheaths & External Constraints — Summed‑Clutch Rotator / Seed

Problem → Dominant pins/sheaths that set the rotation axis are cryptic; multiple weak attachments mask causality • Objective → Identify the minimal anchor set that fixes chirality and gain and validate a multi‑clutch torque‑transmission model • Link → Combinatorial attachment ablations and localized collagenase/perlecanase pulses fitted to a summed‑clutch model

Core intent: Test whether multiple weak, spatially distinct attachments (mesenteric root, sheath seams, ECM contacts) behave as a summed multi‑clutch that sets the hindgut’s rotation axis, chirality, and gain, and whether a minimal anchor subset alone suffices to fix chirality.
