Rate, Time, and Morphogenetic Trajectories — Hindgut / Seed

Problem → Arrhenius scaling breaks near stress‑response thresholds, masking impending failures • Objective → Add non‑Arrhenius gates to forecast safe windows & collapses • Link → Piecewise kinetics with stress‑gated modules (Hsp70, myosin processivity) introduces diagnostic kinks

Core intent: Stress‑gated kinetics create predictable kinks in rate–temperature and phase‑plane plots that reveal safe thermal windows and warn of imminent morphogenetic rate collapse.
