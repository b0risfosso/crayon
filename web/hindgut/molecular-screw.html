The Molecular Screw — Drosophila Hindgut / Seed

Problem → Microscopic chiral biases fluctuate and fail to scale • Objective → Amplify and lock weak bias into tissue-wide twist • Link → Myo1D → Dia arcs → RhoA/ROCK → tilted myosin → adherens coupling

Core intent: show that a weak, noisy cell-level chirality can become a robust, tissue-scale twist if wired into a feed‑forward loop plus neighbor coupling.

1) Noise

Random Dia arcs left/right; scattered RhoA/ROCK; un‑tilted myosin. PCI=0.05, G=0.7, τ=0°.

3) Screw Engages

Tilted junctional myosin → shear aligns neighbor arcs. PCI=0.3, G>1.0.

5) Lock‑in

  • Dia reduced → no arcs, no tilt, τ≈0°.
  • Adherens weakened → local tilt only, mosaic patches, partial τ.
