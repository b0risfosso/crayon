Blueprint to Body Plan — Hindgut Patterning / Seed

Problem → Posterior tissues misinterpret shared signals → ectopic morphogenesis • Objective → Hox‑gated posterior morphogenesis in the hindgut • Link → abd‑A/Abd‑B dosage gates enhancer sensitivity to Torso/Wg/Dpp/Hh, restricting byn/fkh circuits posteriorly

Core intent: Hox dosage gates hindgut enhancers so shared morphogens are interpreted as posterior‑only instructions—licensing elongation and anal valve formation only where abd‑A/Abd‑B permit.
